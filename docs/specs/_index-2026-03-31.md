## Sub-feature Map

### 1. Event Discovery & RSVP
- SF1: Event Feed & Discovery — browsing, filtering, searching events
- SF2: RSVP System — RSVP, capacity, waitlist, QR code confirmation
- SF3: Event Photo Upload — onsite board photo capture, draft event creation
- SF4: Post-Event Materials — file/link uploads after event, attendee access

### 2. Participant Profile & Verification
- SF1: Profile Display — public profile page, avatar, bio, skills, activity summary
- SF2: Profile Editing — edit name, bio, skills, photo upload
- SF3: Trust Score System — scoring rules, history, adjustments, floor of -10
- SF4: Verification Levels — email verified, community verified, badge display

### 3. Skill Exchange Board (Guest Lectures)
- SF1: Lecture Board & Discovery — two-sided board, tabs, filtering, search
- SF2: Posting (Requests & Offers) — university requests, nomad offers, creation form
- SF3: Matching & Application Flow — apply, invite, accept/reject, status lifecycle
- SF4: Feedback System — post-completion ratings, trust score impact

### 4. Campus Facility Booking
- SF1: Facility Directory — browsable catalog, filtering, search, availability
- SF2: Booking Flow — date/time selection, calendar, review, confirm, QR code
- SF3: Cancellation & Trust — cancellation policy, trust penalties, no-show handling
- SF4: My Bookings — upcoming/past bookings, QR access, cancel actions

### 5. Community Discussion Board
- SF1: Forum Feed — thread listing, category filtering, search, pinned threads
- SF2: Thread Creation — create form, rich text, category selection, rate limiting
- SF3: Replies & Discussion — threaded replies, chronological, formatting
- SF4: Thread Interactions — upvotes, bookmarks, best answer marking

### 6. Notifications (LINE & Email)
- SF1: Notification Triggers — 13 event types mapped to channels (Email/LINE)
- SF2: Email Notifications — HTML templates, mock SMTP, QR inline images
- SF3: LINE Notifications — short messages, deeplinks, mock LINE API
- SF4: In-App Notification Center — bell icon, dropdown, full page, preferences

### 7. Non-University Events Organizer
- SF1: Community Event Creation — form, event types, trust gate (>=10), validation
- SF2: Discovery Integration — community badge, feed integration, community filter
- SF3: Organizer Dashboard — event management, attendee list, check-in, announcements

### 8. Manage Recordings (tl;dv)
- SF1: Recording Upload & Linking — tl;dv links, YouTube/Vimeo, direct file upload
- SF2: tl;dv Integration — metadata fetch, transcript, highlights, embed (mock MVP)
- SF3: Recording Library — browsable catalog, filtering, search (incl. transcripts)
- SF4: Viewing Experience — player page, transcript panel, notes, access control
